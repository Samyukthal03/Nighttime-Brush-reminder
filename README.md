#  Brush Time

A beautifully designed, privacy-first nightly ritual alarm clock that ensures you actually brush your teeth before going to bed. The alarm keeps ringing until the camera detects steady, active brushing motion. 

Built entirely in a single HTML file with vanilla JavaScript—no data ever leaves your device.

##  Features

*   **Smart Motion Detection:** Tracks active, localized movement (like a toothbrush stroke) rather than just waiting out a timer.
*   **Encouragement Engine:** Keeps you motivated with positive, rotating text cues while you brush.
*   **Adaptive Theme:** Features a smooth, deep midnight blue and violet backdrop with a dynamic, glowing moon phase indicator that fills up as you progress.
*   **100% Private:** Camera processing happens locally in real-time using HTML5 Canvas. Zero data or video is uploaded to any server.

##  How to Set Up on GitHub Pages

1. **Rename the file:** Make sure your HTML file is named `index.html`.
2. **Create a Repository:** Create a new **Public** repository on GitHub.
3. **Upload:** Upload your `index.html` and this `README.md` to the repository.
4. **Enable Pages:** 
   * Go to your repository **Settings** > **Pages**.
   * Under **Branch**, select `main` (or `master`) and click **Save**.
5. **Get your link:** Wait a minute, refresh the page, and your live URL will appear at the top of the Pages settings menu!

##  Configuration

*   **Ring at:** Set your target bedtime alarm.
*   **Brushing time required:** Adjust the slider to set your required brushing duration. 
*   **Test Now:** Skip the countdown to instantly test the camera sensitivity and audio loop.
